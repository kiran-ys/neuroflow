# 🧠 NeuroFlow — AI-Powered Adaptive Learning Tutor

**Google Solution Challenge 2026 | SDG 4: Quality Education**

> "Every day, 58 students sit invisible while 2 toppers 
> get all the praise. NeuroFlow fixes that."

## 🌐 Live Demo
👉 https://neuroflow-kiran.streamlit.app

## 🎯 Problem
Social media has rewired student brains for instant 
dopamine. Traditional education cannot compete. 
60% of students lose interest in learning — not by 
choice, but by neuroscience.

## 💡 Solution
NeuroFlow uses Google Gemini AI + Flow State Psychology 
+ Game Design to make learning as rewarding as scrolling.

## ✨ Features
- 😊 Mood-aware difficulty adaptation
- 🎯 Auto level detection (Beginner/Moderate/Advanced)
- 🤖 NOVA — AI Coach that speaks at right moments
- 🌊 Real-time Flow Meter
- 🎮 Game-style Level Map
- 🏆 YOU ARE THE TOPPER moment
- 📈 Before/After Transformation Mirror
- 🎊 Confetti + XP reward system

## 🛠️ Built With
- Python + Streamlit
- Google Gemini API (gemini-2.0-flash-lite)
- Flow State Psychology (Csikszentmihalyi)
- Chronobiology + Game Design Principles

pip install -r requirements.txt
